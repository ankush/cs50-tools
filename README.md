# CS50 Tools
###Install Most CS50 tools with few clicks.

###This version includes.
1. CS50 C library.
2. check50 and style50.
3. SPL library

###Installation
You've two options-

1. Download zip and extract it OR clone git ;)    
2. Execute both bash scripts in terminal.   

    sudo bash install.sh
    
    bash setup.sh
